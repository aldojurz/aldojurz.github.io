LENTES PEC


¿QUÉ SIGNIFICA PEC?

Proyecto Escuela Comunidad

Es un proyecto que se llevó a cabo para ayudar economicamente a los alumnos del plantel que necesitan lentes

¿Cómo se organizó?

Los profesores pidieron permiso para llevar a cabo este apoyo con algunas platicas y mostrando como beneficiaria a los alumnos que acepten la ayuda.

→

¿Qué datos pidieron?

Mandaron mensajes a grupos para darles una oportunidad a los alumnos que podrian necesitar esta ayuda, se pidieron datos personales para un mejor orden.

→

¿Cómo fue el examen?

Se pidio prestado un salón para el examen, el proceso no era tradado y llevaba tan solo 2 minutos por persona, se pedian datos, como el nombre y la edad, ademas se ofrecian un diseño para los lentes de tu gusto.

→

Diagnostico socioeconomico

Se contacto al alumno para hacer una encuesta para saber si el apoyo puede ser gratuito o solo dar la mitad del costo de los lentes, dependiendo de su situacion.

→

Entrega

Se dará la entrega en una ceremonia dand a cada alumno despues de haber sido creados todos los lentes.
